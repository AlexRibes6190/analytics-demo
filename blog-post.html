Introducción

Google Analytics 4 es la última versión de la plataforma de análisis web de Google. A diferencia de Universal Analytics, GA4 está diseñado desde cero con un enfoque en eventos y orientado a la privacidad del usuario.

En esta guía completa, te explicaré paso a paso cómo configurar GA4 en tu sitio web usando Google Tag Manager, la forma más profesional y escalable de implementar analytics.

Si prefieres aprender más sobre las diferencias entre UA y GA4, te recomiendo leer nuestra guía de migración de Universal Analytics a GA4.

💡 ¿Sabías que...?

GA4 puede recopilar datos de aplicaciones móviles y sitios web en una sola propiedad.

Crear propiedad GA4

El primer paso es crear una nueva propiedad de GA4 en tu cuenta de Google Analytics. Sigue estos pasos:

  1. Accede a Google Analytics
  2. Haz clic en Admin (esquina inferior izquierda)
  3. En la columna "Propiedad", selecciona Crear propiedad
  4. Introduce el nombre de tu propiedad
  5. Selecciona tu zona horaria y moneda

Instalar código de seguimiento

Una vez creada la propiedad, necesitas instalar el código de seguimiento. La forma recomendada es usar Google Tag Manager:

// En Google Tag Manager
// 1. Crea una etiqueta de tipo "Google Tag"
// 2. Introduce tu Tag ID (G-XXXXXXXXXX)
// 3. Configura el activador en "All Pages"

// El código se verá así en tu página:
gtag('config', 'G-XXXXXXXXXX');

Si prefieres instalarlo manualmente sin GTM, añade este código en el <head> de tu sitio:

<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXXXXXXXX');
</script>
⚠️ Importante: Reemplaza G-XXXXXXXXXX con tu Tag ID real que encontrarás en Admin → Data Streams.

Verificar instalación

Para comprobar que GA4 está funcionando correctamente:

  1. Ve a Admin → DebugView en GA4
  2. Abre tu sitio web en una nueva pestaña
  3. Instala la extensión Google Analytics Debugger
  4. Navega por tu sitio y verifica que aparecen eventos en DebugView

Si ves eventos como page_view, session_start y first_visit, ¡felicidades! GA4 está correctamente instalado.
